<project_specification>
  <project_name>PixelForge - Professional AI Image Generation & Editing Platform</project_name>

  <overview>
    Build a professional-grade, all-in-one image generation and editing platform that rivals Canva, Photopea, and 
    Midjourney combined. The application should provide powerful AI-driven image generation with multiple model 
    providers, a comprehensive canvas-based editing suite with layer management, and advanced AI enhancement tools 
    (upscaling, background removal, style transfer). The platform should be production-ready with a polished UI, 
    comprehensive error handling, and optimal performance for handling up to 4K images.
  </overview>

  <key_interactions>
    <image_generation_flow>
      1. User clicks "Generate" button or uses CMD+G shortcut
      2. Generation modal opens with large prompt textarea
      3. User enters prompt, optionally adds negative prompt
      4. Selects model, style preset, and aspect ratio
      5. Adjusts advanced parameters if needed (steps, guidance, seed)
      6. Clicks "Generate" button
      7. Modal shows progress bar and estimated time
      8. On completion, displays result(s) in grid
      9. User can: use in project, save to library, regenerate, or tweak settings
      10. Click "Use" to add image to canvas as new layer
    </image_generation_flow>

    <canvas_editing_flow>
      1. Image loads on canvas (from generation or upload)
      2. User selects tool from left sidebar
      3. Tool options appear below tool selector
      4. User performs action on canvas (draw, transform, etc.)
      5. Action immediately reflected on canvas
      6. History updates in right sidebar
      7. Layer thumbnail updates
      8. Auto-save triggers after 30 seconds of inactivity
      9. User can undo with CMD+Z or from history panel
      10. Changes persist in project
    </canvas_editing_flow>

    <ai_enhancement_flow>
      1. User selects layer to enhance
      2. Clicks AI tool from left sidebar (e.g., "Remove Background")
      3. If applicable, parameters dialog appears
      4. Confirms action
      5. Progress indicator shows processing status
      6. Preview appears before applying (for upscale, filters)
      7. User accepts or cancels
